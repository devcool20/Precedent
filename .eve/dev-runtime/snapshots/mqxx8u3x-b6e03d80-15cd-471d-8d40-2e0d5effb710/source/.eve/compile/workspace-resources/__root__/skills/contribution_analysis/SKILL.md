# Contribution Analysis Skill

Use these patterns to extract success indicators for contributions:

1. **Commit Hygiene**:
   - Identify format patterns (e.g. Conventional Commits `feat(scope): message` or issue attachments).
   - Trace branch naming models (`fix/description` vs `feat/description`).
2. **PR Structure**:
   - Extract sections from successful PR descriptions (What, Why, Testing done, screenshots).
   - Standardize description templates for copy-paste utility.
3. **Merge Indicators**:
   - Trace average review rounds (e.g. 1 round vs 3+ rounds) and map indicators.
